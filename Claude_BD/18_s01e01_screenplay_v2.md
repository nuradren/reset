## ВИЗУАЛЬНЫЕ/ЗВУКОВЫЕ МАРКЕРЫ

### Три временных слоя:
- **ЛИНИЯ А (репцентр)**: Эхо, металл, гул голосов, казённость
- **ЛИНИЯ Б (Аля)**: Тепло, интимность, творческий хаос, тревожный подтон
- **ЛИНИЯ В (сейчас)**: Чистота, минимализм, пространство

### Переходы:
- Между А и В: глитч (связь через Сашу/Дениса)
- Между Б и остальными: вспышка/затухание (как вспоминание)

### Звук чайника:
- Появляется в каждом временном слое
- Связующий элемент
- "Чайник пережил три переезда, один развод и одну попытку экзорцизма"

---

## РУЖЬЯ, КОТОРЫЕ ВЫСТРЕЛЯТ

1. **Файл hello_world.py** — тайна сезона
2. **"Откуда ты знаешь?"** — вопрос, который задают все клиенты
3. **Аля** — призрак, который вернётся
4. **Солнечное сплетение** — что она сделала?
5. **Друг Дениса** — клиент серии 2
6. **"Попытка экзорцизма"** — упомянуто, но не объяснено
7. **Почему Алекс был в репцентре?** — версия для клиентов vs правда

## СРАВНЕНИЕ С ВЕРСИЕЙ 1.0

| Элемент | Версия 1.0 | Версия 2.0 |
|---------|------------|------------|
| Насилие | Удары | Пощёчины (унизительнее) |
| Поза Алекса | Просто встаёт | Руки в карманах |
| Количество | Неясно | Три (библейский резонанс) |
| Триггер конфликта | Идеология | Орехи (бытовое) |
| Публичность | Упоминается | Колизей-staging детализирован |
| Имена | Кабан/Денис | Два Саши (зеркало) |
| Аля | Отсутствует | Флэшбек + намёки |
| Временные слои | Два | Три |
| Чайник | Упоминается | Сквозной элемент |
